# Entry Server example

이 예제에서는 Entryjs의 API 서버의 예를 보여 줍니다.

## How to use

npm, yarm 또는 pnpm으로 패키지를 설치하고 실행합니다.

```bash
npm install
npm run start
```

```bash
yarn
yarn start
```

```bash
pnpm install
pnpm start
```
