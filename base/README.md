이 예제에서는 Entryjs를 사용하여 화면을 구성하는 예제를 보여줍니다.

## How to use

npm, yarm 또는 pnpm으로 패키지를 설치하고 실행합니다.

```bash
npm install
npm run start
```

```bash
yarn
yarn start
```

```bash
pnpm install
pnpm start
```
